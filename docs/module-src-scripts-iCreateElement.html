(inner) iCreateElement(props) → {void}

For simplify creating custom HTMLElements in DOM
Parameters:
Name Type Description
props object configure your future DOM element
Properties
Name Type Description
tag string any html tag
inner string any html; will append to {tag} you set before
Source:
  • iCreateElement.js, line 3
Returns:
creating custom DOMElement
Type
void
Examples
// void: <body><div><i>my custom HTMLElement is ...</i></div></body>
iCreateElement({})
// void: <body><h3>hello h3</h3></body>
iCreateElement({ tag: 'h3', inner: 'hello h3' })
// void: <body><span class="i-class"><p>any html code</p></span></body>
iCreateElement({
            inner: '<span class="i-class"><p>any html code</p></span>',
        })
